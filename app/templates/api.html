They Work For GoldSU provides a simple GET API which can be used to access site data. At present, it doesn't require authentication and isn't rate limited (so be nice).

The base URL for the API is {{base_url}}

/people

This endpoint exposes all people who currently have accounts as elected representatives.

This endpoint exposes a single user, identified by their ID

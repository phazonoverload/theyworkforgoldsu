About They Work For GoldSU

They Work For GoldSU is an independent platform makes the actions of elected officials at Goldsmiths Students' Union more accessible and transparent.

Historically, Officers of Goldsmiths Students' Union have been asked to provide updates semi-regularly, but these have been inconsistent and hard to find. This platform aims to be the first step in improving this, by giving elected representatives a way to share progress on their promises with the student body.

This can be used both to highlight achievements made, and to allow students to encourage more actions in areas which matter to them.

This project is inspired by MySociety's TheyWorkForYou project, which aims to keep elected MPs accountable. It begun as a piece of coursework for IS52027C: Data, Networks and the Web (2017-18) at Goldsmiths, and is not endorsed by Goldsmiths Students' Union.

All of the code for this project can be found on GitHub with a Creative Commons Attribution-ShareAlike 4.0 International License.
